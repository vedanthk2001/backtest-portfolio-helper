# Brainstorming — Backtest Portfolio Helper

## What We Have Now

- Users input stock tickers + weights (must sum to 100%)
- Backend fetches historical price data from Yahoo Finance (public, no auth)
- Calculates performance metrics: CAGR, volatility, Sharpe ratio, max drawdown
- Results shown across time windows: 1y, 3y, 5y, 10y, max
- Frontend: React + TypeScript + Recharts for visualization
- Backend: Node.js + Express (proxy + compute layer)
